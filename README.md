# Titanic Survival Prediction using XGBoost & LightGBM

 📌 Project Overview
This project applies advanced boosting algorithms (XGBoost and LightGBM) to predict passenger survival on the Titanic dataset.

The objective of this assignment is to preprocess the dataset, train both models, evaluate their performance, and compare their results.


 📂 Dataset
- Titanic_train.csv
- Titanic_test.csv

Target variable:
- Survived (0 = No, 1 = Yes)



🔧 Steps Performed
1. Data Loading
2. Data Cleaning (Handling missing values)
3. Feature Encoding (One-hot encoding)
4. Train-Test Split
5. Model Training:
   - XGBoost
   - LightGBM
6. Model Evaluation using:
   - Accuracy Score
   - Classification Report
7. Performance Comparison



 🤖 Models Used
- XGBoost Classifier
- LightGBM Classifier

 🛠 Technologies Used
- pandas
- numpy
- scikit-learn
- xgboost
- lightgbm
- matplotlib
- seaborn
